To install RoboCrew on your Raspberry Pi, follow these steps:

<Steps>
1. **Set Up Python Environment** (optional but **HIGHLY** recommended):

   ```bash frame="none"
   python3 -m venv robocrew-env
   source robocrew-env/bin/activate
   ```

2. **Install RoboCrew**:

   ```bash frame="none"
   pip install robocrew
   ```

3. **Install Additional Dependencies**:
    For audio support (if using voice commands):

    ```bash frame="none"
    sudo apt install portaudio19-dev
    ```

4. **Configure Udev Rules**:
    Create a new udev rules file for RoboCrew:
    
      ```bash frame="none"
      robocrew-setup-usb-modules
      ```
</Steps>
